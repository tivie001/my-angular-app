Directives

  • Instructions in the DOM
  • Components are directives with a template
  • (i.e. appTurnGreen on an HTML element)
  • a good example of a directive is a ngIf
  • Examples:
    • {{ ngIf }}
    • {{ ngStyle }}
    • {{ ngClass }}
    • {{ ngFor }}

      • {{player}}

Input
directive-comp.ts
            
playerActive = true;
directive.comp.html
            
{{ngIfString}}
Output
Player is: Inactive Active
