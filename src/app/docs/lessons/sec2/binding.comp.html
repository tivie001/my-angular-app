Two-Way-Databinding

  • A way of binding two items together (i.e. input and text) and reacting two event in both directions.

Input
binding-comp.ts
            
playerName = '';
onUpdatePlayerName((event: Event) => { 
this.playerName = (HTMLInputElement(type) event.target).value;
})
binding.comp.html
            
{{ twoWayBind }}
Output


Player Name: {{ playerName }}
